# 4.运算符
1. 基本语法
   - `$(())`或`$[]`
   - expr +、-、\*、/、% 加、减、乘、除、取余
     - 注意:expr运算符间要有空格
2. 案例实操
   - (1) 计算3+2的值
   ```
   [root@Hadoop01 ~]# expr 3 + 2
   5
   ```
   - (2) 计算3-2的值
   ```
   [root@Hadoop01 ~]# expr 3 - 2
   1
   ```
   - (3) 计算(2+3)*4的值
   ```
   [root@Hadoop01 ~]# expr `expr 3 + 2` \* 4
   20
   ```
